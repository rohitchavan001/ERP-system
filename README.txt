ERP SYSTEM - Student Management
================================

Desktop (Windows) + Mobile (Android) with Auto-Sync

QUICK START:
1. Run: BUILD_ALL.bat
2. Install: dist/ERP-System.exe (Desktop)
3. Install: mobile/bin/*.apk (Mobile)
4. Connect: See MOBILE_SETUP_GUIDE.md

FEATURES:
- Desktop EXE with built-in API server
- Mobile APK with offline support
- Auto-sync on same WiFi or internet
- Search, filter, ID cards, reports
